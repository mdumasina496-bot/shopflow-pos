ShopFlow POS — Cashier Quick Guide

ZeHuWo Pty Ltd • Keep this at your till for reference

🧾 Cashier
🔑 Starting Your Shift
  • 1Log in with your username + password
  • 2Go to Admin Menu → Orders
  • 3Enter your password again on the identity screen — this links sales to you
  • 4If "Not Assigned" shows → call your manager to assign you to a till
  • 5Go to Attendance → clock in with your password
🛒 Processing a Sale
  • 1Choose order type: Takeaway / Braai In / Call / Delivery
  • 2Tap product cards to add to cart. Use search bar or category filters
  • 3Adjust qty with + / − buttons. For kg items, type the weight
  • 4For extras (pap, achaar, rolls): tap the Extras tab
  • 5Enter Discount % if applicable
  • 6Press Charge R___ → select Cash / Card / Split
  • 7For cash: enter amount tendered → change shown automatically
  • 8Press Complete → print receipt if needed → New Order
❌ Voiding an Item
  • 1Press VOID next to the item in the cart
  • 2Select the reason from the dropdown
  • 3Call your manager — they must enter their password to approve
⚠️ You cannot void items yourself. Manager approval is always required.
🚪 Ending Your Shift
  • 1Complete or void all open items in cart — never leave items unpaid
  • 2Wait for your manager for cashup
  • 3Attendance → clock out with your password
  • 4Press Logout on the dashboard

⚡ Quick Reference

ShopFlow POS — Manager Quick Guide

ZeHuWo Pty Ltd • Daily operations reference

👔 Manager
🌅 Start of Day
  • 1Log in → Admin Menu → Assign
  • 2Enter manager password → select cashier + till + float amount
  • 3Press Assign — repeat for each cashier
  • 4Check Online Orders — toggle store online if accepting orders
  • 5Clock in via Attendance
💰 End-of-Shift Cashup
  • 1Admin Menu → Store Cashup → + New Cashup
  • 2Select cashier/till — check for abandoned order warning
  • 3Enter card total + petty cash used
  • 4Cashier counts cash by denomination (they cannot see expected total)
  • 5Press Submit → enter your manager password
  • 6Review variance — investigate if not R0
⚠️ Negative variance = cashier is short. Positive = more cash than expected. Both need explanation.
📋 Receiving Stock (GRV)
  • 1GRV → + New GRV → enter supplier + date
  • 2Add each item: product, qty received, cost price per unit
  • 3Press Submit GRV & Update Stock
  • 4For charcoal: GRV → Charcoal/Coal tab → record as "Receive"
  • 5Record charcoal used each braai day: same tab → "Used"
📊 Daily Reports to Check
Today's revenueReports → Order Reports → Daily
Voids todayReports → Order Reports → Voids
Best sellersReports → Order Reports → Best Sellers
Low stock alertReports → Order Reports → Restock
Cashup historyReports → Cashup Reports
⚖️ Stock Count
  • 1Stock Count → Start New Count
  • 2Staff enter counted quantities only — system qty is hidden
  • 3Press Submit → Apply & Save to update stock
  • 4Open the count report to see variances (owner can delete counts)
💡 Never tell staff what the system says before they finish counting.
💬 Online Orders
  • 1Red badge on dashboard = new orders waiting
  • 2Tap Start Processing → Reply on WhatsApp to confirm
  • 3When ready: tap Mark Done
  • 4Adjust wait time in Online Orders → Settings when busy

ShopFlow POS — Owner Quick Guide

ZeHuWo Pty Ltd • Full system control reference

🏪 Owner
👥 Managing Users
  • 1Users → + Add User: name, username, password, role, store
  • 2Tick Module Access to restrict which modules they see
  • 3To disable staff: press Disable on their row
  • 4To change password: Edit → New Password field
🚨 Never share your owner password. Only you can access Users, Activity Monitor, and FC Report.
🔍 Activity Monitor (Audit Trail)
  • 1Admin Menu → Activity Monitor
  • 2See every login, sale, void, GRV, stock count, and override
  • 3Filter by staff member, action type, or date
  • 4"Orders Override" entries = manager accessed Orders for a cashier — investigate
📊 Key Financial Reports
Food cost %FC Report (owner only)
Gross profit/productReports → Stock → GP/FC
Best sellersReports → Order → Best Sellers
Restock neededReports → Order → Restock
Cashup variancesReports → Cashup Reports
Void analysisReports → Order → Voids
⚖️ Stock Count — Owner Powers
  • 1Only YOU can see variance amounts during a count review
  • 2To apply a count: open it → Apply to Stock → owner password
  • 3To delete a wrong count: open it → Delete Count → owner password
🥩 Carcass Breakdown
  • 1Butchery Cuts → + New Record
  • 2Select raw material, enter weight + cost price per kg
  • 3Add each cut: name, weight, selling price per kg
  • 4System shows: yield %, wastage value, gross profit, margin %
🏭 Manufacturing (Wors / Mince / Pap)
  • 1Manufacturing → + New Batch
  • 2Name output product + qty + selling price
  • 3Add input materials → system deducts from stock
  • 4System shows P&L (profit per batch)
